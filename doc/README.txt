ATTENTION: 
To import static data including a page tree an a example configuration, please rename the file ext_tables_static+adt.sql.ATTENTION before the installation to ext_tables_static+adt.sql. Import the static data only in a clear, new system with an empty pagetree, because your pages table is be overwritten. Your DATA WILL BE LOST!

Achtung:
Um statische Daten mit einem Seitenbaum und einer Beispielkonfiguration zu importieren, benennen Sie bitte die Datei ext_tables_static+adt.sql.ATTENTION vor der Installation in ext_tables_static+adt.sql um. Importieren Sie die Daten nur in ein neues System mit einem leeren Seitenbaum, andernfalls können Ihre DATEN VERLOREN GEHEN!
